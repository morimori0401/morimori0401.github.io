军理期末复习

第一章：中国国防

第一节 国防概述

  1. 国防的内涵

    • 国防定义

      国防，是指国家为了防备和抵抗侵略，制止武装颠覆，保卫国家的主权、统一、领土完整和安全所进行的军事及与军事有关的政治、经济、外交、科技、文化、教育等方面的活动。

    • 基本内容

      国防的行为主体为国家，基本内容包括国防建设和国防斗争两个方面。

      • 国防建设是指国家为构建和完善国防体系，提高国防能力而进行的一系列活动的统称。
      • 国防斗争是指国家为了维护安全利益，在军事和与军事相关的政治、经济、外交、科技、社会、心理领域所进行的斗争。

  2. 国防基本类型

    1. 扩张型、自卫型（我国）
    2. 联盟型国防、自主型国防、中立型国防

  3. 国防历史起源

    1. 中国古代国防
      • 始于夏朝
    2. 中国近代国防
      • 从19世纪中叶开始
    3. 中国现代国防
      • 从新中国成立开始
    4. 中国国防历史的启示
      1. 政治昌明是国防巩固的根本
      2. 经济发展是国防强大的基础
      3. 国家统一和民族团结是国防强大的前提

第二节 国防法规

  1. 《国防教育法》的规定
    • 国防教育的方针原则，学校国防教育，社会国防教育，国防教育的保障和法律责任等
  2. 公民国防义务（p16~19）
    1. 兵役义务
      1. 现役
        • 征集年龄：18~24岁
      2. 预备役
        • 征集年龄：18~35岁
        • 第一类：预编到现役部队的预备役士兵；编入预备役部队的预备役士兵；经过预备役登记编入基干民兵组织的人员
        • 第二类：经过预备役登记编入普通民兵组织的人员；其他经过预备役登记确定服士兵预备役的人员
    2. 接受国防教育的义务
    3. 保护国防设施的义务
    4. 保守国防秘密的义务
    5. 协助国防活动的义务
  3. 公民国防权利
    1. 对国防建设提出建议的权利
    2. 对危害国防行为进行制止或检举的权利
    3. 获得经济损失补偿的权利
  4. 权利与义务的关系
    • 对等，平等，同一

第三节 国防建设

  1. 国防战略
    • 概念：国防战略是指筹划和指导国防建设与斗争全局的方略，是国家战略的组成部分。
    • 中国：积极防御战略思想
  2. 国防政策
    • 概念：国防政策是国家制定的一定时期内指导国防活动的基本行动准则。
    • 中国：防御性国防政策
  3. 军民融合
    • 概念：军民融合是我们党按照国防建设和经济建设的紧密相关性，统一富国和强军两大目标，统筹发展和安全两件大事，统合经济和国防两种实力，将有限的社会资源转化为双向互动的生产力和战斗力的战略行动。
    • 总体目标：逐步构建军民一体化的国家战略体系与能力
    • 发展重点：基础设施建设、国防科技工业、武器装备采购、人才培养、军队保障社会化、国防动员等潜力巨大领域与海洋、太空、网络空间等军民共用性强领域。
      1. 推进基础设施统筹建设和资源共享
      2. 推进国防科技工业与武器装备发展
      3. 推进军民科技协同创新
      4. 推进军地人才双向培养交流使用
      5. 推进社会服务和军事后勤统筹发展
      6. 推进国防动员现代化建设
      7. 推进新兴领域军民融合

第四节 武装力量

  1. 中国武装力量的性质
    • 中国人民解放军是中国共产党缔造和领导的，用马克思列宁主义、毛泽东思想、邓小平理论、“三个代表”重要思想、科学发展观和习近平新时代中国特色社会主义思想武装的人民军队。
  2. 中国武装力量构成
    1. 中国人民解放军
      • 现役部队：国家的常备军，主要担负防卫作战任务，必要时可以依照法律规定协助维护社会秩序（p40~45）
        • 陆军、海军、空军、火箭军、战略支援部队、联勤保障部队
      • 预备役部队：努力提高部队的军政素质，不断增强快速动员和整体作战能力；根据未来战争的可能需要，切实做好战时动员的各项准备工作，随时准备转服现役，配合现役部队遂行各种作战任务
    2. 中国人民武装警察部队
      • 中华人民共和国武装力量的重要组成部分，在维护国家安全和社会稳定，保护公民、法人和其他组织合法权益中肩负重大职责
    3. 民兵
      • 不脱离生产的群众武装组织，中华人民共和国武装力量的组成部分，中国人民解放军的助手和后备力量
      • 地方党委、政府和军事系统的双重领导。全国民兵工作由军委国防动员部主管。
      • 分为基干民兵（28岁以下男女公民）和普通民兵（18~35岁男性公民）

第五节 国防动员

  1. 国防动员概述

    1. 产生与发展
      1. 战争动员
        • 动员的规模空前
        • 动员的范围进一步扩展
        • 动员呈现出持续性的特征
      2. 社会主义建设动员
    2. 地位与作用
      1. 遏制战争、打赢战争的基础环节
      2. 应对紧急突发事件的有效措施
      3. 支援经济和社会发展的重要手段

  2. 国防动员的内容

    1. 主要内容

      1. 武装力量动员

      2. 国民经济动员

        • 工业、农业、金融、医疗卫生

        • 财政：国家根据国防需要，将有关财政部门、财政活动及其经济关系由平时状态转入战时状态或紧急状态的活动。

      3. 人民防空动员

      4. 交通战备动员

      5. 政治动员

第二章：国家安全

第一节 国家安全概述

  1. 国家安全的内涵
    • 传统安全->非传统安全
  2. 国家安全的原则
    1. 坚持法治和保障人权
    2. 坚持维护国家安全与经济社会发展相协调，统筹各领域安全
    3. 坚持促进共同安全
    4. 坚持预防为主、标本兼治、专群结合
  3. 总体国家安全观
    • 内容丰富、开放包容、不断发展的安全观念体系
    • 五大要素：以人民安全为宗旨，以政治安全为根本，以经济安全为基础，以军事、文化、社会安全为保障，以促进国际安全为依托
    • 五对关系：
      1. 既重视外部安全，又重视内部安全，强调外部安全与内部安全彼此联系，彼此影响；
      2. 既重视国土安全，又重视国民安全，强调国土安全与国民安全存在有机的统一；
      3. 既重视传统安全，又重视非传统安全，强调传统安全威胁与非传统安全威胁相互影响，并在一定条件下可能相互转化；
      4. 既重视发展问题，又重视安全问题，强调发展和安全是一体之两面，只以其中一项为目标，两个目标军不可能实现；
      5. 既重视自身安全，又重视共同安全，强调全球化和相互以来使得中国和世界的安全已密不可分

第二节 国家安全形势

  1. 我国地缘环境与地缘安全

    • 存在哪些问题？

      • 边界线漫长，相邻国家多，边界争议多

      • 周边地区人口众多，大国数量集中，热点最多

      • 周边国家政治制度及发展水平参差不齐，民族宗教矛盾复杂交织

      • 位于世界两大地缘战略区交接处，面临两个战略方向的挑战

        世界两大地缘战略区：海洋地缘战略区、欧亚大陆地缘战略区

    • 国土面积

      • 陆地边界线：2.2万多千米
      • 领海与内海：38万平方千米
      • 管辖海域（海洋国土）：300多万平方千米

  2. 新形势下国家安全

    • 钓鱼岛争端
      • 背景
        1. 争夺岛礁对改善日本地缘环境的意义非同一般
        2. 岛礁周围丰富的石油、天然气、矿产渔业资源对日本有战略意义
        3. 美国始终是钓鱼岛争端背后的推手
    • 南海争端
      • 实质：一些国家非法侵占我国南沙群岛部分岛礁引起的领土争议，以及南海沿岸国家之间的海洋权益主张重叠问题
      • 属于我国的海域：
        • 东——黄岩岛
        • 西——南沙群岛万安滩
        • 南——南沙群岛曾母暗沙
        • 北——东沙群岛北卫滩
    • 中印边境争议
      • 东段争端
        • 传统习惯线与麦克马洪线之间的争端
      • 西段争端
        • 新疆阿克赛钦地区归属之争
      • 中段争端
        • 西藏阿里地区与印度马偕尔邦和北方邦接壤的边界地区

  3. 国家安全新兴领域

    1. 网络空间安全
      1. 网络安全事件频繁发生
      2. 网络主权之争日趋强烈
      3. 网络军备竞赛逐步升级
    2. 外层空间安全
      1. 外层空间处于和平开发、利用和非军事化的状态
      2. 本国航天器在外层空间不受干扰、碰撞或攻击威胁的状态
    3. 国际海底区域安全
    4. 极地安全
    5. 生物安全

第三节 国际战略形势

  1. 国际战略形势现状与发展趋势
    1. 世界多极化进程加快
    2. 经济全球化进程出现波折
    3. 大国战略博弈更加激烈
    4. 全球性问题和挑战空前增多
      1. 恐怖主义威胁不减
      2. 网络日益成为犯罪分子甚至恐怖组织的活动平台
      3. 愈演愈烈的难民问题
      4. 大国在战略军备与军控领域博弈加剧
  2. 相关国家军事力量及战略动向
    1. 美国
    2. 俄罗斯
    3. 日本
    4. 印度
      1. 军事力量
        1. 核大国
        2. 军力部署
        3. 当今前五大军费开支国
      2. 战略动向
        1. 防范重点：巴基斯坦、中国
        2. 采取措施
          1. 以国际合作带动创新，推进国防工业建设
          2. 动态调整军事战略与作战思想
          3. 不断推进战略武器建设和增强联合作战能力
