绪论

  • 模式识别：采集、描述、识别、决策

​

  • 模式识别的常用范式：
  1. 句法：每个模式都可认为是一个句子，关系可以认为是语法
    • 优点：适合结构性强的模式
    • 缺点：抗噪声能力差，计算复杂度高
  2. 模板匹配
    • 优点：直接简单
    • 缺点：适应性差
  3. 统计方法：根据训练样本训练一定的模式识别机器
    • 统计决策理论：根据每一类总体的概率分布决定决策边界
    • 判别分析方法：给出边界，根据某种准则，由训练样本决定最优参数
  4. 神经网络：学习、推广、自适应等
    • 优点：可以有效地解决一些复杂的非线性问题
    • 缺点：缺少有效的学习理论

  • 监督模式识别与非监督模式识别

    • 监督模式识别：可以获得一定数量的类别已知的样本
    • 非监督模式识别：考察样本间的相似性
    • 半监督模式识别

  • 训练与调试

    • 欠拟合：模型过于简单以至于不能反映出该类样本的全部特性

      ​

    • 过拟合：模型过于复杂，反映了数据的一些无关特性（如噪声）

​

贝叶斯决策

  • 基本原理：

    1. 决策规则：后验概率归一化（除以P(x)）后比较

    2. 后验概率->先验概率 × 类条件概率 / （所有求和）先验概率 × 类条件概率

      【其中类条件概率是在曲线上查的】

    3. 似然比

    4. 负对数（将相乘变成相加）

      【注意：分子分母与符号都发生变化】

    5. 要求：各类别的条件概率、先验概率、总体概率密度已知

  • 类条件概率密度函数与后验概率图像

    • 后验概率图线交点为决策边界或者分类线
      • 类条件概率密度函数图像与后验概率图像的交点不相同
      • 如果两类分类中第二类的类条件概率变大，则决策边界左移

  • bayes最小错误率决策

    • 条件错误率：一个分类的错误率
    • 错误率：整体的错误率
    • 贝叶斯决策：一致最优决策

  • 最小风险贝叶斯决策

    • 引入风险函数（损失函数，决策与自然状态的函数），可以用决策表来表示。

    • 决策表是人为设定的，决策表不同，决策结果就不一样。

    • 计算过程：

      1. 计算后验概率（需要先验概率与类条件概率）
      2. 用后验概率乘以权重，计算条件风险（注意这里用的是决策错误的后验概率来乘权重）
      3. 对k个条件的风险值进行比较，找出风险最小的决策a

  • 分类器的设计（小结）

    • 确定判别函数：对于每个类别，都有一个判别函数
    • 决策面：通过联立两个决策面函数找出来

  • 正态分布时的统计决策

    • 原因：正态分布有较好物理意义，数学计算简单
    • 协方差矩阵：表示各个维度之间两个变量之间的关系
      • 数据上两个维度并不是完全不相关的，有一定的正负相关性，但并不是一直线性相关的
    • 等密度点的轨迹为一个超椭球面，中心由μ决定，主轴方向由矩阵的本征向量决定，主轴长度与本征值正相关
    • 等密度点的轨迹的马氏距离相等【马氏距离体现类条件概率】

    ​

    • 多元正态：判别函数与决策面（第二章ppt，p51-56）

      ​ 几种特殊情况：

      1. 不同类别之间先验概率不等——此时是线性分类器，先验概率大的一类有更大的决策空间

      2. 不同类别之间先验概率相等——此时是最小距离分类器，超平面与两类别均值的连线正交并且通过连线中点

      3. 各类的协方差矩阵都相等——各类别的样本集中在以该类别均值为为中心的同样形状的超椭球内

        【进一步】当先验概率相等时可以简化为马氏距离。决策规则：只需要计算样本x到各类的均值向量的马氏距离平方，然后把x归为具有最小距离的这一类。

      4. 一般情况下，分布仍然是一个超平面，过x0但是不与两类别均值的连线正交

        【如果各类先验概率相等，x0为两类别均值的连线的中点；各类先验概率不等，决策面向先验概率较小的方向偏移】

      5. 各类协方差不相等——判别函数为x的二次型，决策面为超二次曲面，决策面的不同由方差项的差异引起【见p65】

  • 重要：第二章ppt第72面例题

概率密度函数的估计

  • 由训练样本集估计总体概率密度：
    • 监督参数估计：同类样本来估计本类的类条件概率密度
    • 非监督参数估计：已知总体概率密度函数形式，推断出参数
    • 非参数估计
  • 参数估计的基本概念
    1. 统计量：针对不同要求构造出样本的某种函数
    2. 参数空间：总体概率密度函数形式已知，未知分布中的参数θ，所有θ的取值构成参数空间
    3. 点估计：构造一个统计量作为参数θ的估计
    4. 估计量：点估计得到的量称为估计量
  • 先验概率估计
  • 类条件概率密度估计（重点）
