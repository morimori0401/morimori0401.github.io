求最大正应力（8.26）

  1. 利用力之间关系，得出静不定方程

  2. 利用杆的变形条件（一般是总Δl=0），得出约束方程

  3. 解原静不定方程

  4. 求最大力，从而求最大正应力

求许用扭力偶（9.20）

  1. 利用力偶之间关系，得出静不定方程

  2. 利用力偶与扭角之间的关系（一般是两处转动角度相等），选取恰当参考面，得出约束方程

  3. 根据许用切应力的公式，求出M的取值范围

剪力-弯矩方程/图

一般来说，剪力方程是弯矩方程的导数

弯曲强度与弯曲刚度

  • 抗弯截面系数决定梁的弯曲强度。

  • 惯性矩决定梁的弯曲刚度。

  • 纵向正应变*弹性模量E=弯矩M/抗弯截面系数Wz

求最大弯曲拉应力与弯曲压应力

  1. 求截面上的弯曲内力
  2. 建立参考坐标系，确定截面形心与中性轴的位置
    • 确定形心的方法：部分的面积与部分的形心的乘积之和/全部的面积
  3. 计算截面对中性轴z的惯性矩
  4. 依据公式σ=M*y/Iz，计算最大拉/压应力
    • 最大拉、压应力应该发生在上下边缘处
    • 具体发生在哪个边缘处要具体问题具体分析

【11.19】

边界条件与连续条件

  • 边界条件：边界处挠度与转角为0
  • 连续条件：梁分段处变形左右相等
  • 挠曲轴拐点：弯矩为0的点

【12.22】

  • 可以用某一点的力矩平衡来验证受力图对不对。一般用力的作用点来判断

  • 要记得通过力的方程与力矩的方程来求未知力。在格式上，用ΣFx、ΣFy、在B点的

  • 力偶矩不限于一定的作用点。因此有力偶矩的题目，可以灵活地选用作用点，并不一定要在题上力偶的作用点

  • 在算分布载荷的力矩的时候，可以用积分，也可以直接用集中载荷处理

  • 活动铰链一个约束：Fy

  • 杆一个约束：F

  • 固定铰链两个约束：Fx、Fy

  • 固定端有三个约束：Fx、Fy、M

  • 问体系中有几个约束，则是问可能的力与力矩有几个。

【以下都是对单个点来分析的】

  • 平面汇交力系：2个方程

  • 平面任意力系：3个方程（刚体可以看作这个）

  • 平面力偶系：F=0

  • 空间汇交力系：3个方程

  • 空间任意力系：6个方程

  • 空间平行力系：3个方程

  • 桁架：杆数为m，提供支撑的部分的约束为a，节点数为n，满足m+a<=2n则为静定

  • 判断零力杆：节点法，如果有一个方向的力为0，那么该方向的杆为零力杆

  • 截面法：尽量选取通过杆较少的截面。先整体后分离体

  • 拉伸、扭转、弯曲

  • 线性阶段、屈服阶段、硬化阶段、缩颈阶段——低碳钢拉伸性能曲线

  • 强度

  • 刚度

  • 稳定性

【12.25】

材料力学：连续性假设、均匀性假设、各向同性假设

内力：向外（使其受拉）为正（轴力，剪力，扭矩，弯矩）

基本思想

  • 分段求内力
  • 分段求变形（几段变形求和）
  • 叠加法：几个载荷同时作用的总效果，等于分效果之和

胡克定律

  • 应力：F/A
  • 应变：应力/E
  • 变形：Δl = Fl/EA

圣维南原理

静定与静不定问题

节点位移法

  • 取随便一点（注意不要取在任意梁的延长线上），对两梁所在直线作垂线，垂足与该点原来所在位置的距离则为变形距离（这里用垂线来近似）

  • 条件：刚性xx、固定xx（不能变形，可以从长度或者角度来列方程）

  • 首先画出变形后图，再根据该图画出简要受力图。注意力的朝向要严格按照变形的方向来画

  • 具体解法

    1. 静力学平衡方程（根据受力分析作出，活用力平衡方程与力矩平衡方程）
    2. 物理方程（Δl = Fl/EA，注意E跟A有可能不同，所以要分别列出）
    3. 变形协调方程（见图上几何关系，不仅可以用长度，还可以用角度）

  • 相关题型：围绕许用应力来出

  • 注意正应力、正应变与切应力、切应变的区别与联系

  • 对于拉压杆正应力公式F/A，应力不一定要在比例极限范围（这里的比例极限范围指的是应变=应力/E）内。

  • 应力状态分析

  • 复杂应力状态

扭转

  • 矢量方向作用为拉的时候为正。（图上会给旋转的方向，用右手螺旋法则变化成拉压之类的作用）

  • 切应力与ρ成正比，方向垂直于半径。

  • Ip与Wp需要记住。有空心与实心的区别

【12.26】

扭转相关的基本公式

  • 强度校核：Tmax/Wp

  • 刚度校核：T/GIp = dφ/dx

  • Ip = πD^4/32，Wp = πD^3/16（实心条件下）

  • Ip = πD^4(1 - α^4)/32，Wp = πD^3(1 - α^4)/16

  • M = 9549P/n（n为转速）

  • T = P/ω

  • 结论：同样条件下设计轴，空心比实心重量小

扭转的静不定问题

  • 两端固定的扭转问题解法

    • 注意：设正法
    • 静力学平衡（从M入手，类似受力分析）
    • 几何方程（以其中一个固定端入手，其转角为0）
    • 物理方程（代入以上相关公式，注意用φ表示转角）

  • 注意：在变截面轴算τ的时候，不能直接代入最大的

  • 切应力互等定理：在微体的互垂截面上，垂直于截面交线的切应力数值相等，方向都指向或者离开交线

  • 应力集中：构件靠近凹槽的时候，圆角的局部范围内应力急剧增大。主要影响脆性材料。

  • 扭转没有正应力：纯剪切应力状态

  • 泊松效应：拉伸时，管的半径会减小，但是不同部分减小得不一样

弯曲

  • 剪力：使顺时针转动的为正（左上右下）
  • 弯矩：顶部受压为正
  • 中间节点：不能传递力偶，只能传递力
  • 纯弯梁假设：弯曲平面假设+单向受力假设

剪力方程、弯矩方程

  1. 求支反力（这一步很重要，一定要做！）

  2. 建立坐标

    1. （截面法，看除了关键点之外的面，此时设立剪力与弯矩为正）建立剪力、弯矩方程（要把方程本身列出来！！）
    2. （面积法：q向上为正，M顺时针为正）最右端的力与矩可以作为验证

  4. 画剪力弯矩图

  • 如果仅受分布载荷，最大弯矩发生在分布载荷的合力作 用

  • 可以适当建立局部坐标系，有时候会让问题变得简单

【12.27】

  • 平面假设、单向受力假设
  • 中性层：既不伸长也不缩短的面
  • 中性轴：没有正应力。非纯弯的时候有切应力，且切应力为最大。
  • 抗弯截面系数Wz

步骤

  1. 建立几何方程
  3. 静力学方面

计算静矩与形心（回去要看）

  • 形心计算：负面积法（空出来的面积当成负数来算，其余与一般形心计算相同）
  • 惯性矩计算（算出形心之后）（平行移轴定理）：若是矩形，用bh^3/12+Aa^2（此处a为到形心移动的距离）
    • 【注意】原来的轴应该是该物体的形心轴
    • 不能用负面积法

梁的强度条件

  • 正应力小于许用正应力

  • 切应力小于许用切应力

  • 步骤：

    1. 内力分析：危险面
    2. 画剪力弯矩图
    3. 应力分析：危险点
    4. 强度校核

  • 塑性材料与脆性材料不同。

  • 合理安排约束、合理安排加载方式、加配重、变截面梁与等强度梁

弯拉组合

  • 一个偏心压缩应力=一个沿轴应力+一个弯矩

弯曲位移

  • 计算梁位移的叠加法【载荷叠加、变形叠加（需分段刚化） 】
    • 熟记1，3，4，6，8，9挠度与转角（对应M0，F与q）
    • 利用对称性求挠度与转角

弯曲静不定方程

  • 梁的刚度条件：许用挠度与许用转角（与扭转问题的强度与刚度类似）一般挠度与转角都要叠加计算，所以没有那么直接
  • 载荷集度大于零
  • 低碳钢是塑性材料，铸铁为脆性材料（更抗压）

【12.28】

应力状态分析

  • 解析法
  • 应力圆
    • 点面对应
    • 转向对应
    • 二倍角对应
  • 主应力排序：按照代数值排序：σ1≥σ2≥σ3
  • 注意三向应力圆的画出方式：如果跟其中一个应力平行，则不受其影响

广义胡克定律

  • 条件：材料各向同性+处于线弹性范围

强度理论

  • 适用于脆性材料（铸铁）
    • σr1 = σ1（第一强度理论）
    • σr2 = σ1 - μ(σ2 + σ3)（第二强度理论）
  • 适用于塑性材料（低碳钢）
    • σr3 = σ1 - σ3（第三强度理论）
    • σ4 =1/√2 * √(σ1 - σ2)+()+()

复杂应力状态的强度问题（非常重要，现在还不怎么会）

  • 外载分解：分解为拉压、扭转和弯曲组合
  • 内力计算
  • 应力分析
  • 强度计算
    • 注意：铸铁的拉跟压的许用应力不一样，要注意应用与计算

薄壁圆筒的应力分析

  • 轴向
  • 周向

压杆稳定

  • 高阶解的意义：多了约束
  • 两端非铰支细长压杆的临界载荷
    • 增加约束，μ会减小
